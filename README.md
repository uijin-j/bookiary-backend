  <summary>📂 서비스 주요 기능</summary>

  ### Book
    - 책 검색하기 (with Naver API)
    - 내 책장에 저장하기 (북마크)
    - 내 책장에 있는 책에 '좋아요' 하기
    - 내 책 공유하기 (with 노트 공유 / 시간 제한)

  ### Member
    - 카카오 소셜 로그인 / 회원가입

  ### Note
    - 내 책장에 있는 책에 노트 작성하기
    - 내가 남긴 노트 조회하기
    - 내가 남긴 노트 수정하기
    - 내가 남긴 노트 삭제하기
